# Pico Word Clock

# Hardware

* Raspberry Pi Pico 2 W
* Pixels Strip WS2812B
* Small breadboard

# Notes
* 11 x 10 Grid. 110 pixels.
* Even rows red left to right. Odd rows opposite.
